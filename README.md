# AI Interview Preparation and Coding Practice Platform

A next-generation, responsive AI platform designed to help users prepare for technical interviews through real-time simulations, code execution, and resume analysis.

## Project Structure

This is a decoupled monorepo containing:
- `/client`: Next.js frontend application.
- `/server`: Node.js Express backend API.

## Features (Planned)
- **Authentication & Security**: Clerk or Firebase Auth.
- **AI Resume Analyzer**: PDF parsing and feedback.
- **Dynamic Interview Simulator**: Conversational AI interviewer.
- **Integrated Developer Environment (IDE)**: Split-screen code editor with execution.
- **Real-Time Voice Interviews**: Speech-to-text pipeline for audio input.

## Tech Stack
- Frontend: Next.js (React), Tailwind CSS.
- Backend: Node.js, Express.js.
- Database: PostgreSQL, Prisma ORM.
- AI: `@google/genai` or `openai` SDKs.
